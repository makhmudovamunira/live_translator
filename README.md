# 🎤 AI Voice Translator

Real-time Uzbek → English voice translator built with FastAPI.

## Features

- 🎤 Speech Recognition
- 🌍 Uzbek → English Translation
- 🔊 English Voice Output
- ⚡ FastAPI Backend
- 🖥 HTML + JavaScript Frontend

## Technologies

- FastAPI
- SpeechRecognition
- Google Translator
- gTTS
- HTML
- JavaScript

## Installation

```bash
git clone https://github.com/USERNAME/translator.git

cd translator

pip install -r requirements.txt

uvicorn main:app --reload
```

Open

```
http://127.0.0.1:8000
```
